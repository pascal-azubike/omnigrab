Get Started

Up in 30 seconds.

🪟
Windows
  1. 1 Download the .exe installer above
  2. 2 Run the installer (click "More info" → "Run anyway" if SmartScreen prompts)
  3. 3 Launch OmniGrab from the Start menu
🍎
macOS
  1. 1 Download the .dmg disk image above
  2. 2 Open it and drag OmniGrab to Applications
  3. 3 First launch: right-click → Open to bypass Gatekeeper
🐧
Linux
  1. 1 Download the .AppImage above
  2. 2 Make it executable: chmod +x OmniGrab*.AppImage
  3. 3 Run it: ./OmniGrab*.AppImage
🤖
Android
  1. 1 Download the .apk above
  2. 2 Enable "Install from unknown sources" in Android Settings
  3. 3 Tap the APK to install, then open OmniGrab
